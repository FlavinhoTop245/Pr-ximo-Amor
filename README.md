# ❤️ Próximo Amor

**Muito mais que vagas. Uma ponte de impacto social.**

O **Próximo Amor** é uma plataforma inovadora que conecta talentos temporários e voluntários qualificados a empresas e projetos sociais. Nosso objetivo é transformar o voluntariado em uma ferramenta de desenvolvimento mútuo, onde empresas fomentam impacto social e profissionais ganham experiência real e networking.

---

## 🚀 Visão Geral

O projeto atua em duas frentes principais:

- **Para Empresa/Organizações:** Encontre profissionais dedicados para projetos pontuais, fortalecendo a cultura de responsabilidade social e desenvolvendo *soft skills* na equipe.
- **Para Voluntários:** Uma oportunidade de aplicar conhecimentos técnicos em causas reais, construir um portfólio sólido com avaliações e expandir sua rede profissional.

## ✨ Funcionalidades

- 🔍 **Busca Inteligente de Vagas:** Filtre oportunidades por área de atuação e localização.
- 🎯 **Match de Afinidade:** Visualização da porcentagem de compatibilidade entre o voluntário e a vaga.
- ➕ **Criação de Oportunidades:** Interface intuitiva para empresas cadastrarem novas vagas de impacto.
- 👤 **Perfil Profissional:** Gerenciamento de informações do voluntário e histórico de contribuições.
- 💬 **Mensagens & Notificações:** Sistema de comunicação direta entre interessados e organizadores.
- 📱 **Interface Responsiva:** Design moderno e amigável para diversos dispositivos.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído com as melhores tecnologias do ecossistema front-end:

- **React 18**: Biblioteca principal para construção da interface.
- **Vite**: Build tool extremamente rápida para desenvolvimento moderno.
- **Supabase**: Backend-as-a-Service para banco de dados e autenticação.
- **Lucide React**: Biblioteca de ícones elegantes e leves.
- **React Router 6**: Gerenciamento de rotas e navegação na aplicação.
- **Vanilla CSS**: Estilização personalizada focada em performance e design premium.

## ⚙️ Configuração do Ambiente

Siga os passos abaixo para rodar o projeto localmente:

### 1. Pré-requisitos
- [Node.js](https://nodejs.org/) (versão 18 ou superior)
- npm ou yarn

### 2. Instalação
Clone o repositório e instale as dependências:

```bash
git clone https://github.com/FlavinhoTop245/Pr-ximo-Amor.git
cd Pr-ximo-Amor
npm install
```

### 3. Variáveis de Ambiente
Crie um arquivo `.env` na raiz do projeto com as credenciais do seu projeto Supabase:

```env
VITE_SUPABASE_URL=seu_url_do_supabase
VITE_SUPABASE_ANON_KEY=sua_chave_anon_key
```

### 4. Execução
Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

O projeto estará disponível em `http://localhost:5173`.
